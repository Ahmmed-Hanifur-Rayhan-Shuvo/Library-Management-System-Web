# Library Management System - Web Version

A complete web-based library management system converted from a Python Tkinter application. This system provides all the functionality of the original Python application in a modern, responsive web interface.

## 🌟 Features

### User Portals
- **Student Portal**: Borrow books, return books, check fines, view available books
- **Faculty Portal**: Extended borrowing periods, course management, research access
- **Librarian Portal**: Full system administration, user management, book inventory

### Core Features
- 📚 Book management (Add, Update, Remove, Search)
- 🔄 Borrow and return tracking
- 💰 Automated fine calculation
- 👥 User registration and management
- 📊 Reports and statistics
- 🔐 Secure authentication system
- 📱 Fully responsive design

## 🚀 Quick Start
