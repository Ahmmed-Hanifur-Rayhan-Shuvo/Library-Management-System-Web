System Features

Comprehensive library management capabilities

Book Management

Add, update, remove, and search books with detailed information including author, publisher, and availability status.

Borrow & Return

Streamlined borrowing and returning process with automatic availability updates and transaction records.

Fine Calculation

Automated fine calculation based on borrowing duration with different rules for students and faculty.

User Management

Register and manage students, faculty, and librarians with role-based access control.

Reports & Analytics

Generate reports on book usage, user activity, and financial transactions for better decision making.

Secure Authentication

Secure login system with encrypted passwords and session management for all user types.

About This System

This Library Management System is a web-based application designed to digitize and streamline library operations. Originally developed in Python with Tkinter GUI, this web version provides enhanced accessibility and user experience.
